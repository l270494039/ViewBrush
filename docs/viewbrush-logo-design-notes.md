# ViewBrush Logo Design Notes

本文件用于整理 `ViewBrush` 首轮品牌 logo 提案的设计说明，方便内部讨论、方向筛选与后续 refinement。

## 1. 品牌基础判断

`ViewBrush` 的品牌气质不应落入以下常见方向：

- 通用 AI 工具品牌
- 宠物电商品牌
- 可爱型插画品牌
- 纯科技感视觉系统

结合当前网站设计与品牌故事，logo 需要服务的核心认知是：

- `AI gives you clarity. Human hands give it soul.`
- 先让用户“看清”作品方向，再由真实手工完成最终画作
- 品牌像策展型艺术服务，而不是效率型工具
- 视觉上要传达温暖、可信赖、可收藏、可上墙

因此，logo 的方向应优先强调以下四点：

- `View`: 看见、预览、框定、清晰判断
- `Brush`: 手作、笔触、工艺、人的完成度
- `Gallery`: 可陈列、可收藏、具有作品属性
- `Warmth`: 温暖、家居感、高级但不冷

## 2. 统一视觉原则

本轮 5 个方案虽然符号语言不同，但共享同一套基础规则：

- 配色以暖米白、深棕、克制金棕为主
- 避免蓝紫色科技感、霓虹感和夸张渐变
- 避免直接使用宠物头像、爪印等低门槛宠物符号
- 保持 `vector-friendly`，便于后续转为正式品牌标准稿
- 在网站导航、包装、证书、角标、小尺寸 favicon 中都应有识别度

当前主色倾向可参考：

- `Deep Brown`: `#241C16` / `#2D241B`
- `Warm Ivory`: `#F6F0E7`
- `Muted Gold`: `#A58964`

## 3. 本轮设计目标

这次不是直接做最终定稿，而是先验证品牌符号语言应该往哪里走。

首轮需要回答的核心问题有三个：

1. `ViewBrush` 更适合“字标型品牌”还是“符号型品牌”？
2. 品牌核心应该更偏“画廊感 / 出版感”，还是更偏“工坊感 / 徽章感”？
3. “View” 与 “Brush” 的结合，是应该通过字母、框景、视窗，还是通过印章系统来表达？

## 4. 五个方向说明

### 4.1 方案一：Frame Monogram

文件：

- `output/imagegen/viewbrush-logo-concepts/01-frame-monogram.png`

设计概念：

- 以字母 `V` 为主角
- 外轮廓借用了画框或作品标签的边角结构
- 在 `V` 的一侧加入带笔触感的收尾，连接 `View` 与 `Brush`

希望传达的感受：

- 稳定
- 高级
- 经典
- 画廊感

优点：

- 最像成熟艺术品牌
- 适合放在网站 navbar、包装角标、落款位置
- 小尺寸下依然容易识别
- 后续很适合发展成标准化 monogram 系统

潜在问题：

- 叙事偏克制，品牌故事需要靠文案补足
- 如果后续字体处理不够精细，可能会略保守

适合方向：

- 如果希望品牌显得更稳、更成熟、更像艺术机构，这个方向非常合适

### 4.2 方案二：View Aperture + Brush

文件：

- `output/imagegen/viewbrush-logo-concepts/02-view-aperture-brush.png`

设计概念：

- 用四个角构成“被框定的观看区域”
- 中间用斜向笔触建立“刷痕”与“创作动作”
- 视觉结构更强调“先看清，再下笔”

希望传达的感受：

- 清晰
- 现代
- 有判断感
- 品牌解释力强

优点：

- `View` 与 `Brush` 的语义结合最直接
- 相比纯字标，更容易形成独立品牌图标
- 在网站、社媒头像、favicon 中应用潜力较高

潜在问题：

- 如果后续处理过硬，容易稍微靠近现代设计咨询或创意工作室
- 需要控制好笔触质感，避免过于“设计软件化”

适合方向：

- 如果希望品牌更年轻一点、更有解释力，同时保留高级感，这个方向值得深入

### 4.3 方案三：Editorial Wordmark

文件：

- `output/imagegen/viewbrush-logo-concepts/03-editorial-wordmark.png`

设计概念：

- 以 `ViewBrush` 文字本身作为品牌主角
- 在字母细节中加入轻微手作变化
- 用极小的辅助图标支撑完整识别系统

希望传达的感受：

- 出版感
- 文艺
- 精致
- 克制

优点：

- 很符合当前网站偏 editorial、gallery 的视觉系统
- 字标本身高级感强，适合大部分网页和品牌展示场景
- 对品牌名称本身有放大作用，有利于记忆完整拼写

潜在问题：

- 过度依赖字体质量和字间关系
- 独立图标记忆点会弱于 monogram 或 seal
- 小尺寸 favicon 需要另做辅助图形系统

适合方向：

- 如果品牌希望更像艺术出版品牌、画册品牌、策展型品牌，这个方向最贴切

### 4.4 方案四：Gallery Seal

文件：

- `output/imagegen/viewbrush-logo-concepts/04-gallery-seal.png`

设计概念：

- 用圆章式结构建立“证书感”和“工坊印记感”
- 内部结合画框轮廓与画笔元素
- 强化“作品完成度”和“手工背书”的品牌印象

希望传达的感受：

- 收藏感
- 工艺感
- 仪式感
- 可信赖

优点：

- 很适合包装、证书、吊牌、封签、品牌章
- 让品牌更像有制作传统与品质承诺的工作室
- 在实体接触点里会很有说服力

潜在问题：

- 比前几个方向更“重”
- 若细节控制不好，容易显得偏传统
- 网站导航中的轻盈感可能不如字标方向

适合方向：

- 如果品牌后续会非常强调“手工完成”“证书感”“礼物属性”，这个方向很适合作为辅助品牌章，甚至可与主 logo 组合使用

### 4.5 方案五：Window Portal

文件：

- `output/imagegen/viewbrush-logo-concepts/05-window-portal.png`

设计概念：

- 以视窗 / 展墙 / 门廊感的外形构成品牌符号
- 将笔触置于“被观看的空间”之中
- 表达“把记忆转化为可陈列作品”的过程

希望传达的感受：

- 空间感
- 策展感
- 家居陈列感
- 转化与展示

优点：

- 与品牌故事“上墙”“展示”“空间归属”非常一致
- 图形有较强记忆点
- 在网站 hero、包装和空间展示场景中都容易建立联想

潜在问题：

- 方向更概念化，成败很依赖比例和线条精度
- 如果处理不够收敛，可能与某些家居或空间品牌语汇靠近

适合方向：

- 如果希望品牌更强调“作品属于空间”而不是“工具属于流程”，这个方向很值得继续打磨

## 5. 五个方向的对比总结

如果按“品牌角色”来理解，本轮五个方向可以这样看：

- `方案一` 更像品牌签名与主角标
- `方案二` 更像品牌方法论的图形表达
- `方案三` 更像品牌名字本身的高级化处理
- `方案四` 更像品牌章与品质背书系统
- `方案五` 更像品牌故事的空间化象征

如果按当前网站的适配度来判断：

- 与 navbar、页脚、网页排版最契合：`方案一`、`方案三`
- 与品牌故事表达最契合：`方案二`、`方案五`
- 与包装证书和工艺感最契合：`方案四`

## 6. 当前推荐

如果本轮目标是先选出最有潜力的主方向，我建议优先看这三个：

- `方案一 Frame Monogram`
- `方案三 Editorial Wordmark`
- `方案五 Window Portal`

推荐原因：

- `方案一` 最稳，最有成熟品牌相
- `方案三` 最符合现有网站调性
- `方案五` 最有故事性和品牌独特性

如果需要一个更强的图标型方向，可以把 `方案二` 作为重点备选。

## 7. 下一轮 refinement 建议

一旦确定其中 1 到 2 个方向，下一轮建议进入更细的品牌精修，而不是继续广泛发散。

建议 refinement 内容：

1. 调整字标字体风格与字距比例
2. 统一主 logo、icon、small-size 版本的结构关系
3. 补齐深底、浅底、纯黑白版
4. 验证在 navbar、favicon、包装标签中的可读性
5. 最终收敛为可转矢量的正式品牌稿

## 8. 一句话总结

`ViewBrush` 的 logo 设计不应强调“AI 生成”，而应强调“被看清的创作方向”与“被手工完成的作品价值”。

这也是本轮所有方案的共同出发点。
